# dotfiles

## Setup repo

### 01. Clone repository

```
$ git clone git@github.com:eisyos/dotfiles.git
```

### 02. Change directory

```
$ cd dotfiles
```

### 03. Exuecute setup.sh

```
$ ./scripts/setup.sh
```

## How to use set up script for tmux

### 01. Start tmux

```
$ tmux
```

### 02. Execute the script

```
$ ./tmux/setup.sh
```
